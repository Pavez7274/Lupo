# Coffee
> Eval, Js, Ts, Bash

Evaluate Javascript, Typescript Or Coffeescript Code Via `Eval()`, `Typescript.Transpile()`, `Coffee.Compile()` Or `Child_process Execsync()`

## Fields

| Name | Type | Required |
|------|------|----------|
| Code | String | Yes |

## Command Usage
```
?coffee <code>
```
