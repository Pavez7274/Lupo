# Eval
> Coffee, Js, Ts, Bash, Exec

Evaluate Javascript, Typescript, Coffeescript Or Bash Code Via `Eval()`, `Typescript.Transpile()`, `Coffee.Compile()` Or `Child_process.Execsync()`

## Fields

| Name | Type | Required |
|------|------|----------|
| Code | String | Yes |

## Command Usage
```
?eval <code>
```
